# Instructions

This document outlines the required setup and daily assignments for the AI Agents course. Complete the following setup before starting the course.

##### 1. Kaggle Account
- Create a Kaggle account.
- Learn how Kaggle Notebooks work.
- **Phone verify your account** (required for all codelabs).

##### 2. AI Studio
- Create an AI Studio account.
- Generate an API key.

##### 3. Kaggle Discord
- Create a Discord account.
- Join the Kaggle Discord server to collaborate with other learners.

# Course Schedule

## Day 1 — Introduction to Agents

### Overview

This unit introduces AI agents, including:

- Taxonomy of agent capabilities
- AgentOps for reliability and governance
- Agent interoperability
- Agent security and identity

### Codelabs

- Build your first AI agent using **Gemini + ADK**
- Build your first multi-agent system using **ADK**
- Use **Google Search** within your agents for up-to-date information

### Assignments
